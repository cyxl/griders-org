              IDs

              To get someones discord ID go to discord settings, go to advanced, then turn on develepor mode. Right click someones icon to copy their id.


              If you want null(s) in any of the lists, just press enter or click the arrow button in any list input boxes that doesn't any character(s) in it.

              All of the things below, except "After", need to have something in it.

              Supported escape characters in builder.html:

              \b, \t, \n, \f, \r, and \u{0000-FFFF}


              Token

              The token of the bot you're using (put your account token in if using a self bot, actual bot recommended) To make a bot (recommended) go here -> https://youtu.be/k5BspQ9Czek?t=19To find token go here -> https://www.youtube.com/watch?v=LnBnm_tZlyU (must be in browser)


              Permissions

              The ID of who the bot will listen to, you can have null(s) in this list, if left blank or null the script will not check.


              Bomb messages


              Random

              Character length of generated gibberish for random messages. (recommended 10)(cpu intensive)

              Fixed

              This list is used by the fixed bomb command, you cannot have any null in here. (Discord things like ** and @everyone do work)


              Webhook spams


              Usernames

              The bot wil randomly pick the items entered in this list and use them as the username of the webhook message. You cannot have any null(s) in this list.

              Profile pictures

              The bot will randomly choose an image URL from the list for the profile picture of a webhook, you can have null(s) in this list, which will appear as a blank pfp.To get the url of an image go to a browser and rightclick an image, then press open image in new tab. Paste the address in the search bar and put that here.

              Contents

              The bot will randomly choose a message and use it as the message of a wbhook, you may not have null(s) in this list. (@everyone and discord text things work)


              Bot permissions

              The permissions a bot gets when invited into a server, not needed for self bots.


              Command prefix

              The prefix you put before a message so a bot will listen to a command. (ex !, .)


              On start status

              The status at which the bot will appear, Invisible, DND, Idle, Online.


              Verbose levels

              (How much it tells you it does in the terminal) It is recommended you turn on all (15), but for a really low quality pc you can turn off ones you find unnessacary.


              Ban whitelist

              If you don't want ban all commands to affect a certain user put their id here.


              After

              You can later modify things with the config command, to see more information, start the bot and type the config command with no suffix.


              .config command

              Everything in here can be modify later with the .config command. To see more informations on how to use .config command, you have to use run the .config after booting up the bot.
